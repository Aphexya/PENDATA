Local Outlier Factor (LOF)#

Local Outlier Factor (LOF) adalah metode yang digunakan untuk mendeteksi outlier dalam dataset berdasarkan densitas lokal. LOF membandingkan kerapatan suatu titik data dengan tetangga terdekatnya untuk menentukan apakah titik tersebut merupakan outlier.

~ Contoh menghitung LOF terdapat 10 Data dengan 2 Fitur yakni SL & SW

~ Langkah 1 - Menghitung Encludian Distance

Cara menghitungnya yakni hitung jarak antara tiap pasangan titik. Ambil contoh kita mau hitung jarak 1 dan 2 dengan rumus berikut :

Kita presentasikan SLa dan SWa adalah data dari ID 1 yg akan kita hitung dengan tiap pasangan titik, untuk SWa dan SWb adalah data dari ID 2.

Jadi d(1,2) = √ (5,1 - 4,9)^2 + (3,5 - 3,0)^2 hasilnya adalah d(1,2) = 0,539

Selanjutnya lakukan hal yg sama dengan tiap titip

~ Langkah 2 - Menentukan K-Distance dan K-Nearest Neighbors

Cara menentukannya adalah yg pertama mencari K-Nearest terlebih dahulu dengan cara mencari 3 data yg paling kecil dari deretan tiap ID. Ambil contoh disini saya mau mencari 3 data yg kecil dari ID 1

Pertama kita ambil deretan ID 1 dari hasil perhitungan Euclidean Distance tadi, selanjutnya urutkan nilai yg paling kecil ke besar. Lalu ambil 3 data yg paling kecil, jadi hasilnya adalah 0.141 (5), 0.141 (8), 0.447 (10) itulah hasil dari K-Nearest yakni [5, 8, 10]

Selanjutnya cara menentukan K-Distance yakni kita ambil data K = 3 dari K-Nearest tadi yakni urutan ke 3 dari data yg paling kecil, jadi di ID 1 urutan ke-3 nya adalah 0.447 (10)

Lakukan hal yg sama di beberapa titik ID

~ Langkah 3 - Cara menghitung Reachability Distance (RD)

Cara menghitung RD adalah dengan mencari nilai maksimal dari K-Distance tiap K-Nearest ID dengan Euclidian ID dan K-Nearest, yakni dengan rumus berikut :

Ambil contoh disini saya mau hitung ID 1 dengan titik K-Nearest [5, 8, 10] kita hitung tiap K-Nearest nya agar menghasilkan RD yakni titik K-Nearest [5]

reach-distk(1, 5) = MAX(0,447, 0,141) hasilnya adalah 0,447

Lakukan hal yg sama tiap titik K-Nearest

~ Langkah 4 - Cara menghitung Local Reachability Density (LRD)

Cara menghitung LRD ialah jumlah k (yakni disini adalah k = 3) dibagi dengan RD antara titik K-Nearest dan tetangga nya yakni 3 tetangga yg terdekat, rumusnya berikut :

Ambil contoh disini saya mau menghitung LRD dari ID 1 dengan K-Nearest [5, 8, 10] kita ambil titik ini

LRD(1) = 3 / (0,447 + 0,316 + 0,447) hasilnya adalah 2,479

Lakukan hal yg sama di beberapa titik

~ Langkah 5 - Cara menghitung LOF Score

Cara menghitung LOF yakni tiap 3 titik LRD(B) dibagi dengan LRD(A) dan dibagi dengan jumlah k tetangga (yakni k = 3)

Jadi LRD(B) presentasi dari 3 titik tetangga K-Nearest dari hasil LRD tadi, sedangkan LRD(A) presentasi dari titik ID dari LRD. Rumusnya berikut :

Ambil contoh disini saya mau hitung LOF ID 1 yakni

LOF(1) = { (2,459/2,459) + (2,459/2,459) + (3,571/2,459) } / 3 Hasilnya adalah 1,151

Lakukan hal yg sama di beberapa titik ID

Jadi Outliernya adalah titik ID 6 dana 9 yakni 1,344 dan 1,228

Karena LOF nya lebih besar dari 1 dan tidak terlalu dekat dengan 1

import numpy as np
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
from sklearn.neighbors import LocalOutlierFactor

# Data contoh (10 data dengan 2 fitur)
data = np.array([
    [5.1, 3.5], [4.9, 3.0], [4.7, 3.2], [4.6, 3.1], [5.0, 3.6],
    [5.4, 3.9], [4.6, 3.4], [5.0, 3.4], [4.4, 2.9], [4.9, 3.1]
])

# Inisialisasi model LOF dengan k=3
lof = LocalOutlierFactor(n_neighbors=3, metric="euclidean")

# Fit model dan dapatkan skor LOF (semakin negatif, semakin outlier)
lof_scores = lof.fit_predict(data)
lof_values = -lof.negative_outlier_factor_

# Buat DataFrame untuk melihat hasilnya
df = pd.DataFrame(data, columns=["SL", "SW"])
df.insert(0, "ID", range(1, 11))  # Menambahkan kolom ID dari 1-10
df["LOF"] = lof_values
df["Outlier"] = ["Ya" if x > 1.2 else "Tidak" for x in lof_values]

# Tampilkan hasil
print(df)

# Visualisasi data
plt.figure(figsize=(8, 6))
sns.scatterplot(x=df["SL"], y=df["SW"], hue=df["Outlier"], palette={"Ya": "red", "Tidak": "blue"}, s=100)
plt.title("Deteksi Outlier dengan LOF")
plt.xlabel("SL")
plt.ylabel("SW")
plt.show()
