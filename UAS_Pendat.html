#Klasifikasi Kanker Payudara Menggunakan Algoritma Data Mining pada Dataset Breast Cancer Wisconsin (Diagnostic)

#📘Pendahuluan – Breast Cancer Wisconsin (Diagnostic) Dataset

Kanker payudara merupakan salah satu penyakit yang paling umum dialami oleh wanita di seluruh dunia. Deteksi dini terhadap kanker payudara sangat penting dalam meningkatkan peluang kesembuhan dan mempercepat penanganan medis. Salah satu metode utama dalam diagnosis kanker payudara adalah melalui pengamatan karakteristik sel dari hasil biopsi jaringan.

Dengan perkembangan teknologi data mining dan machine learning, diagnosis kanker payudara dapat dilakukan secara otomatis menggunakan data hasil observasi medis. Salah satu dataset populer yang digunakan dalam bidang ini adalah Breast Cancer Wisconsin (Diagnostic), yang berisi data karakteristik statistik dari inti sel kanker berdasarkan hasil observasi mikroskopik.

Melalui penerapan algoritma klasifikasi pada dataset ini, diharapkan dapat dibangun model prediktif yang mampu membedakan antara kanker payudara jinak (benign) dan ganas (malignant) dengan akurasi tinggi, sehingga dapat mempercepat proses diagnosis dan mengurangi potensi kesalahan manual dalam dunia medis.

#📊 Data Understanding – Breast Cancer Wisconsin (Diagnostic)

##Sumber Data Dataset diambil dari link berikut ini:

🔗 https://archive.ics.uci.edu/dataset/17/breast+cancer+wisconsin+diagnostic

Dataset Breast Cancer Wisconsin (Diagnostic) tersedia di UCI Machine Learning Repository dan berasal dari data hasil biopsi digitized fine needle aspirate (FNA) pada massa payudara. Dataset ini terdiri dari total 569 observasi, yang masing-masing menggambarkan pengukuran statistik dari sel kanker.

Terdapat 30 fitur numerik yang mewakili berbagai karakteristik inti sel, seperti:

  • Mean radius, texture, perimeter, area, smoothness

  • Standard error dari setiap karakteristik

  • Nilai maksimum (“worst”) dari setiap karakteristik

Label target dari setiap data adalah:

  • M (Malignant/Ganas)

  • B (Benign/Jinak)

Sehingga dataset ini sangat cocok untuk digunakan dalam studi klasifikasi biner.

##Tujuan

Tujuan dari analisis ini adalah untuk mengembangkan model klasifikasi yang mampu membedakan jenis kanker (ganas vs jinak) berdasarkan karakteristik sel yang telah diukur. Model prediktif yang dihasilkan akan membantu proses diagnosis kanker payudara secara otomatis, meningkatkan efisiensi klinis, serta mendukung pengambilan keputusan medis yang lebih akurat dan cepat.

📌 Fitur pada Dataset Breast Cancer Wisconsin (Diagnostic)

Fitur adalah variabel independen yang digunakan sebagai input untuk memprediksi label diagnosis (ganas atau jinak).

Dataset ini memiliki 30 fitur numerik yang berasal dari pengukuran karakteristik inti sel kanker dari hasil biopsi. Masing-masing fitur dihitung berdasarkan tiga grup statistik: mean, standard error, dan worst (nilai maksimum).

Berikut contoh 10 fitur penting yang umum digunakan dalam klasifikasi:

🔬 Karakteristik Statistik Inti Sel:

  1. radius_mean Rata-rata jarak dari pusat ke batas inti sel. Ukuran ini berkaitan dengan ukuran sel secara keseluruhan.

  2. texture_mean Ukuran variasi tingkat intensitas pada permukaan inti sel (gradasi abu-abu pada gambar mikroskop).

  3. perimeter_mean Keliling inti sel yang dihitung berdasarkan kontur sel pada gambar.

  4. area_mean Luas inti sel yang diamati dari gambar hasil biopsi.

  5. smoothness_mean Mengukur tingkat ketidakteraturan permukaan sel. Nilai kecil menunjukkan permukaan halus, sedangkan nilai tinggi menunjukkan bentuk bergerigi atau tidak rata.

  6. compactness_mean Menggabungkan perimeter dan area untuk menggambarkan kerapatan inti. Nilai tinggi bisa menunjukkan bentuk sel yang tidak normal.

  7. concavity_mean Ukuran tingkat cekungan pada batas sel. Sel kanker biasanya lebih cekung dibandingkan sel normal.

  8. concave_points_mean Jumlah titik-titik cekung yang terbentuk di batas sel. Fitur ini penting untuk membedakan sel ganas dan jinak.

  9. symmetry_mean Ukuran seberapa simetris bentuk inti sel terhadap garis tengahnya.

  10. fractal_dimension_mean Menggambarkan kompleksitas batas sel menggunakan dimensi fraktal. Sel kanker cenderung memiliki batas yang lebih kompleks.

#⚙️ Preprocessing Data – Breast Cancer Wisconsin (Diagnostic) Dataset

Pada tahap preprocessing, dilakukan normalisasi data menggunakan StandardScaler untuk menyamakan skala antar fitur numerik yang memiliki rentang nilai berbeda-beda, seperti radius_mean, area_worst, hingga concavity_se.

Proses imputasi nilai hilang (missing values) tidak diperlukan karena dataset Breast Cancer Wisconsin (Diagnostic) tidak mengandung nilai kosong atau NaN.

Deteksi outlier dilakukan sebagai bagian dari eksplorasi data, terutama karena beberapa fitur seperti area_worst atau radius_mean memiliki nilai ekstrem. Namun, penghapusan outlier tidak dilakukan agar informasi yang mungkin signifikan secara medis tetap dipertahankan.

Proses seperti binning dan reduksi dimensi tidak dilakukan karena:

  • Semua fitur bersifat numerik kontinu.

  • Tidak terdapat fitur kategorikal yang perlu dikonversi.

  • Tidak ada kebutuhan khusus untuk memvisualisasikan dalam dimensi rendah pada tahap awal analisis.

##Normalisasi

# --- PREPROCESSING: Normalisasi Breast Cancer Wisconsin Dataset ---

import pandas as pd
from sklearn.preprocessing import StandardScaler
from ucimlrepo import fetch_ucirepo

# 1. Ambil dataset Breast Cancer Wisconsin (Diagnostic) dari UCI
bcw = fetch_ucirepo(id=17)

# 2. Gabungkan semua data (fitur + label)
data = bcw.data.original.copy()

# 3. Hapus kolom 'ID' (tidak relevan)
if 'ID' in data.columns:
    data.drop(columns='ID', inplace=True)

# 4. Encode label 'Diagnosis': M → 1 (malignant), B → 0 (benign)
if 'Diagnosis' in data.columns:
    data['Diagnosis'] = data['Diagnosis'].map({'M': 1, 'B': 0})
else:
    raise ValueError("Kolom 'Diagnosis' tidak ditemukan!")

# 5. Pisahkan fitur dan label
X = data.drop(columns='Diagnosis')
y = data['Diagnosis']

# 6. Normalisasi fitur menggunakan StandardScaler
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# 7. Konversi kembali ke DataFrame
X_scaled = pd.DataFrame(X_scaled, columns=X.columns)

# 8. Gabungkan kembali dengan label untuk keperluan analisis
data_scaled = X_scaled.copy()
data_scaled['Diagnosis'] = y

# 9. Simpan ke file CSV (opsional)
data_scaled.to_csv("breast_cancer_normalized.csv", index=False)

# 10. Tampilkan ringkasan hasil normalisasi
print("✅ Data Breast Cancer berhasil dinormalisasi:")
print(X_scaled.describe())

#🧪 Modelling – Breast Cancer Wisconsin (Diagnostic) Dataset

Klasifikasi kanker payudara dilakukan menggunakan lima algoritma klasifikasi yaitu:

  1. K-Nearest Neighbors (KNN)

  2. Naive Bayes

  3. Decision Tree

  4. Logistic Regression

  5. Random Forest

Tujuan dari penggunaan kelima algoritma ini adalah untuk membandingkan performa masing-masing metode dalam mendeteksi jenis kanker (ganas atau jinak) secara dini berdasarkan 30 fitur numerik yang tersedia pada dataset Breast Cancer Wisconsin.

Setiap model dievaluasi menggunakan metrik:

  • Akurasi (accuracy)

  • Presisi (precision)

  • Recall

  • F1-score

Langkah ini bertujuan untuk mengetahui seberapa efektif model dalam memprediksi jenis kanker dengan benar, terutama untuk kasus ganas (yang berdampak kritis).

##Persiapan (Data Split & Normalisasi)

Data di-split agar model bisa dilatih pada sebagian data (training) dan diuji pada data lain yang belum pernah dilihat (testing), sehingga evaluasi lebih objektif dan mencerminkan kemampuan model dalam memprediksi data baru.

import pandas as pd
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler
from ucimlrepo import fetch_ucirepo

# Ambil data
bcw = fetch_ucirepo(id=17)
data = bcw.data.original.copy()

# Hapus kolom ID jika ada
data.drop(columns='ID', inplace=True, errors='ignore')

# Encode diagnosis: M → 1, B → 0
data['Diagnosis'] = data['Diagnosis'].map({'M': 1, 'B': 0})

# Pisahkan fitur dan label
X = data.drop(columns='Diagnosis')
y = data['Diagnosis']

# Normalisasi fitur
scaler = StandardScaler()
X_scaled = scaler.fit_transform(X)

# Split data
# Split data menjadi data latih dan data uji
# Tujuannya agar model dapat dilatih pada sebagian data (training set)
# dan dievaluasi secara objektif pada data yang belum pernah dilihat (testing set),
# sehingga performa model mencerminkan kemampuannya dalam generalisasi
X_train, X_test, y_train, y_test = train_test_split(
    X_scaled, y, test_size=0.2, random_state=42
)

##1️⃣ K-Nearest Neighbors (KNN)

from sklearn.neighbors import KNeighborsClassifier
from sklearn.metrics import classification_report, accuracy_score, confusion_matrix
import seaborn as sns
import matplotlib.pyplot as plt

# 1. Inisialisasi dan latih model
knn = KNeighborsClassifier()
knn.fit(X_train, y_train)

# 2. Prediksi
y_pred_knn = knn.predict(X_test)

# 3. Evaluasi
print("📊 K-Nearest Neighbors")
print("Accuracy:", accuracy_score(y_test, y_pred_knn))
print(classification_report(y_test, y_pred_knn, target_names=['Benign', 'Malignant']))

cm = confusion_matrix(y_test, y_pred_knn)

plt.title('Confusion Matrix - KNN')

##Tabel Perbandingan Evaluasi Semua Model

from sklearn.metrics import accuracy_score, precision_score, recall_score, f1_score

# Buat fungsi evaluasi
def evaluate_model(y_true, y_pred):
    return {
        "Accuracy": accuracy_score(y_true, y_pred),
        "Precision": precision_score(y_true, y_pred),
        "Recall": recall_score(y_true, y_pred),
        "F1-Score": f1_score(y_true, y_pred)
    }

# Evaluasi semua model
results = {
    "K-Nearest Neighbors": evaluate_model(y_test, y_pred_knn),
    "Naive Bayes": evaluate_model(y_test, y_pred_nb),
    "Decision Tree": evaluate_model(y_test, y_pred_dt),
    "Logistic Regression": evaluate_model(y_test, y_pred_lr),
    "Random Forest": evaluate_model(y_test, y_pred_rf),
}

# Konversi ke DataFrame
results_df = pd.DataFrame(results).T  # Transpose supaya model jadi baris

# Format 4 angka desimal
results_df = results_df.round(4)

# Tampilkan tabel evaluasi
print("📋 Evaluasi Model Breast Cancer Dataset:")
results_df.insert(0, 'No.', range(1, 1 + len(results_df)))
print(results_df)
